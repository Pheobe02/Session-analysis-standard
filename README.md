# Chat4Data Session Analysis Standard

This repository stores the reusable standard for daily Chat4Data online user session analysis.

The standard is intentionally separate from the Chat4Data product codebase. It can later evolve into:

- A standalone GitHub repository.
- A daily manual analysis checklist.
- An automation that reads exported session CSV files and writes issue-session details to Feishu documents.

## Documents

- [线上用户会话分析标准](docs/session-analysis-standard.md)

## Daily Outputs

Each daily analysis should produce two outputs:

1. Session analysis summary report.
2. Issue-session detail table.

All metrics and classifications should follow the standard document.
